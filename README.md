## 0. 개요 및 목적

여태까지 자신이 만든 프로젝트와 코드를 업로드 할 수 있는 게시판이다.

포트폴리오 작성을 어려워하는 사람들을 위해 디자인적 편의 제공과 정보 공유가 주 목적이다. 

## 1. 메뉴 구조도

![menu](https://user-images.githubusercontent.com/97935451/206866448-b9bd4209-9895-4963-979e-15f7748a8f51.PNG)

## 2. 주요 기능

1. 회원가입 및 로그인, 로그아웃
2. 게시글 작성, 삭제, 수정<br>
(3. 포트폴리오 작성, 삭제, 수정)
4. 댓글 작성, 삭제, 수정<br>
(5. 이미지 업로드)<br>
6. 회원 정보 관리

## 3. 개발환경 및 DBMS

오라클 클라우드 서버
 - MySQL
 - 아파치 톰캣 9.0
 - 자바 17
 - (Spring 예정)
